Candidate Tracker Instructions

The purpose of this system is to keep track of job candidates: their contact information, their status and next action, any associated documents such as resumes, interview schedules, and comments. From the home page of the Tracker, you can add new candidates to the system, search for existing candidates, and, if authorized, add and change information about candidates.

Candidate Lifecycle

Each candidate goes through a lifecycle, represented in the Tracker by the "Next Action" value associated with each job candidate.  The description and order of the next-action values is entirely configurable and by default is as follows, in order of occurence in the lifecycle:

none No next action is planned or is intended to be planned.  This is functionally equivalent to "no value" which is represented as the string "-- NONE --" in wherever the next action items appears in the user interface.
preliminary phone screen The next action is for HR or the recruiter to telephone the candidate to determine if he is still available, interested, and able to work in the USA.
functional resume review The next step is for someone in the hiring department to read the candidate's resume to assess whether or not the candidate is qualified for the position.
hold There is no next step planned, but some new "next action" will be assigned to this candidate in the near future.
technical phone screen The next action is for someone in the hiring department to telephone the candidate to assess if an on-site interview is warranted.
schedule first round Having determined that the candidate warrants an on-site interview, the next step is to schedule that interview.
complete first round The first round has been scheduled but has not completed yet.
schedule next round If, after the first round, it is decided to hold a subsequent round, the next action is to schedule it.
complete next round Once a subsequent round has been scheduled, the next action is to complete it.
refer to CS This candidate should be referred to Customer Services
check references After successful interviews, reference checks are made.
create offer Once a decision has been made to offer a position to the candidate, an offer letter must be generated.
send reject note Otherwise, a rejection letter should be sent.
close The offer letter has been presented.  The next step is to close, or sign,  the candidate
start work The offer letter has been signed, next step is to start work.

Tracker Actions

Add a Candidate

From the home page, click the "Add a Candidate" link.  The candidate entry page appears.  Enter any information that is known about the candidate tha tyou have.  Select which Job Opening the candidate has applied for, assign an owner, and select a Next Action which is likely to be "preliminary phone screen" or "functional resume review."Click the "Save" button.

Return the the candidate page and upload any documents you have on the candidate such as a cover letter and resume.

Query

From the home page, check the "Query" link.  The main query page appears.  Select search parameters from the lists and optionall type a few character to use in a name match.  Hit the "Submit" button.  A list if results will appear.  You can sort on most of the column headings and go to a candidate information page by clicking on the candidate's name.
